在本目录执行以下命令即可运行 CLI（控制台仅输出统计信息，具体结果会写入 JSON，默认文件名为 seed_results.json）：

1. 直接随机生成（默认模式为 random，与游戏一致）：
   dotnet run --project SeedRollerCli.csproj -- --count 20 --character ironclad --ascension 0 --result-json my_results.json

2. 使用配置文件（例如 config.json）：
   dotnet run --project SeedRollerCli.csproj -- --config config.json

常用可选参数：
- --seed-info seed_info.json      （可选）仅在测试自定义 seed_info.json 时指定绝对路径；默认优先使用程序集内置或程序目录下的同名文件。
- --result-json seed_results.json 覆盖结果输出路径；若省略则写入默认文件。
- --mode incremental              如需旧版“顺序递增”种子，可显式设置该参数，并配合 --start-seed。
- --start-seed 0000000000         指定顺推起点，random 模式下可省略。
- --character defect              选择角色（ironclad/silent/regent/necrobinder/defect）。
